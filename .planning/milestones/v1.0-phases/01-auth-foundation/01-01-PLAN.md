  <files>

  <action>
    Make three targeted edits to wire the Google Services Gradle plugin — same pattern as puzzlenook.

    **android/build.gradle.kts** — The current file uses `allprojects {}` but has no `buildscript {}` block. Add a `buildscript` block at the top (before `allprojects`):

    ```kotlin
    buildscript {
        repositories {
            google()
            mavenCentral()
        }
        dependencies {
            classpath("com.google.gms:google-services:4.4.0")
        }
    }
    ```

    **android/app/build.gradle.kts** — Add `id("com.google.gms.google-services")` to the existing `plugins {}` block, after the Flutter plugin line:

    ```kotlin
    plugins {
        id("com.android.application")
        id("kotlin-android")
        id("dev.flutter.flutter-gradle-plugin")
        id("com.google.gms.google-services")  // ADD THIS
    }
    ```

    **android/app/src/main/AndroidManifest.xml** — Add a `<queries>` entry for Google Play Services and a `<meta-data>` entry inside `<application>`. The manifest already has a `<queries>` block for PROCESS_TEXT — add a new `<package>` entry to it:

    Inside the existing `<queries>` block, add:
    ```xml
    <package android:name="com.google.android.gms" />
    ```

    Inside the `<application>` tag (after the existing `flutterEmbedding` meta-data), add:
    ```xml
    <meta-data
        android:name="com.google.android.gms.version"
        android:value="@integer/google_play_services_version" />
    ```

    Do NOT modify any other parts of these files.
  </action>
